About Analyses

Analyses allow you to compare multiple laps to find improvements in your driving.

How it works:
  1. Create an analysis with a descriptive name
  2. Add laps from your sessions
  3. View comparison charts to see differences
  4. Identify where you can improve
Tips:
  • Compare laps from the same track/car for best results
  • Use descriptive names like "Baseline vs New Setup"
  • Track and car fields help you filter laps later
