How to Use
Step 1: Download the Client

Download the iRacing Telemetry Client for Windows from our GitHub Releases page.

Step 2: Configure the Client
  1. Run the client and right-click the system tray icon
  2. Click "Settings"
  3. Paste your API token into the "API Token" field
  4. Set the server URL to: ws://{{ request.get_host }}/ws/telemetry/live/
  5. Click "OK" to save
Step 3: Start Streaming
  1. Launch iRacing
  2. Right-click the client system tray icon
  3. Click "Start Streaming"
  4. Join a session in iRacing
  5. Your session will appear in the Live Sessions page
